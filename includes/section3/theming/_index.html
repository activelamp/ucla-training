Template structure of a page

Top most template: html.html.twig

  • Show the html.html.twig file in stable

Page Top and Page Bottom are special

These sections are not regions

Template structure of a page (cont)

Main Body: page.html.twig

  • Show the page.html.twig file in stable

Template structure of a page (cont)

Regions: region.html.twig

region--[region].html.twig

Blocks have templates

Nodes have templates

Views have templates

Anything that renders HTML should have a template

All templates can be overridden in theme.

Read more about theme templates here: https://www.drupal.org/docs/8/theming/twig/twig-template-naming-conventions
