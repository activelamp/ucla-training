Twig Basics

Printing a Variable


{# Print a string #}
{{ foo }}

{# Print object property #}
{{ bar.baz }}

Twig will check:


// Array key.
$foo['bar'];
// Object property.
$foo->bar;
// Object method.
$foo->bar();
// Object get method convention.
$foo->getBar();
// Object is method convention.
$foo->isBar();
// Object dynamic object property is set and get property.
`$foo->__isset('bar');`
`$foo->__get('bar');`

Filters


{# Print a string #}
{{ foo | upper }}

{# Print entire object except baz property #}
{{ bar | without('baz') }}
