Infrastructure

Applications should manage all dependencies

These are all dependencies of the app!

In true 12-factor, we would define the configuration for each of these dependencies of the app, and the entire app on our local development machines would be no different than what is in production. If you remember that was Factor 10. We aren't going to deep into configuring infrasructure. Instead, we're going to use a system that gives us some sensible defaults for all of these dependencies.
